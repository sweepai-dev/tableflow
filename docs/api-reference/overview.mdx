TableFlow API Overview

TableFlow is an open-source CSV importer that provides a simple and efficient way to import and manage CSV files.

Features:
1. Import: The Import feature allows users to easily import CSV files into the application. It supports various file formats and ensures data integrity during the import process.
2. Template: The Template feature allows users to create and manage templates for CSV files. It provides a flexible way to define the structure of the CSV files and ensures consistency across different imports.
